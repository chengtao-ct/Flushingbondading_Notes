## 核心术语速查

| 术语 | 英文 | 简要说明 |
|------|------|---------|
| 时域 | Time Domain | X轴时间、Y轴电压的分析方式 |
| 波形 | Waveform | 电信号随时间变化的图形 |
| 格 | Division/div | 屏幕基本单位，代表的值由设置决定 |
| 时基 | Time/Div | 每格代表的时间 |
| 带宽 | Bandwidth | 示波器能准确测量的最高频率 |
| 采样率 | Sample Rate | ADC每秒采样次数 |
| 存储深度 | Memory Depth | 能存储的采样点总数 |
| 触发 | Trigger | 告诉示波器"何时开始显示"的条件 |
| 触发电平 | Trigger Level | 触发条件对应的电压阈值 |
| 上升时间 | Rise Time | 信号从10%到90%幅度的时间 |
| 峰峰值 | Vpp | 波谷到波峰的总电压差 |
| 有效值 | Vrms | 等效做功能力的直流电压 |
| 占空比 | Duty Cycle | 脉宽占周期的百分比 |
| 耦合 | Coupling | DC/AC/GND，决定什么信号进入示波器 |
| 探头补偿 | Probe Compensation | 校准探头与示波器的匹配 |
| 混叠 | Aliasing | 采样不足导致高频信号伪装成低频 |
| 振铃 | Ringing | 信号边沿后的阻尼振荡 |
| 毛刺 | Glitch | 短暂的非预期电压尖峰 |
| BNC | Bayonet Neill-Concelman | 示波器的同轴连接器接口 |
| 通道 | Channel | 独立输入通道，每通道测一个信号 |
