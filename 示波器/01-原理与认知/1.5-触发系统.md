## 核心要点

> 触发告诉示波器**"从哪个时刻开始画波形"**，让每次画图起点一致，波形就稳定不动。触发 = 触发源 + 触发条件 + 触发模式。

---

## 1. 触发的工作原理

示波器不是"实时"显示的，而是不断"拍照-显示-拍照-显示"循环。触发决定了每次"拍照"的起始时刻。

```
无触发：波形在屏幕上左右乱跑（每次起点不同）
有触发：波形稳定不动（每次起点一致，波形重叠）
```

**类比：** 拍一匹奔跑的马——没有触发，每张照片马在不同位置；有触发，每张照片马在正中央。

---

## 2. 触发三要素

| 要素 | 含义 | 选择原则 |
|:---|:---|:---|
| **触发源（Source）** | 用哪个通道触发 | 选你最关心的信号通道 |
| **触发条件（Condition）** | 什么条件触发 | 边沿触发最常用（90%） |
| **触发模式（Mode）** | 等不到触发怎么办 | 日常用 Auto，捕获偶发用 Normal/Single |

---

## 3. 边沿触发（Edge Trigger）——90%的场景

**条件：** 信号在指定方向穿越触发电平的那一刻。

```
触发源 = CH1
触发沿 = 上升沿
触发电平 = 1.5V

→ 每当 CH1 信号从低向高穿过 1.5V 时，示波器开始采集
```

### 触发电平（Trigger Level）

旋转 **Trigger Level 旋钮** 调节，屏幕右侧有三角形标记（▶）指示当前位置。

| 操作 | 效果 |
|:---|:---|
| 顺时针 | 触发电平上移（电压增大） |
| 逆时针 | 触发电平下移（电压减小） |

**最佳位置：信号幅度的 50% 处。**

| 信号类型 | 幅度 | 推荐触发电平 |
|:---|:---|:---|
| 3.3V 方波 | 0~3.3V | 1.5V ~ 1.65V |
| 5V 方波 | 0~5V | 2.0V ~ 2.5V |
| 市电 220V | ±311V | 0V |

### 触发沿选择

| 场景 | 推荐 |
|:---|:---|
| 时钟信号 | 上升沿 |
| PWM 信号 | 上升沿 |
| 下降沿触发的协议 | 下降沿 |
| 上升沿不稳定时 | 试试下降沿 |

---

## 4. 三种触发模式

| 模式 | 触发条件满足 | 触发条件不满足 | 适用场景 |
|:---|:---|:---|:---|
| **Auto** | 稳定显示 | 强制刷新（波形漂移） | 日常调试 |
| **Normal** | 稳定显示 | 保持上一帧（屏幕"死"了） | 捕获间歇信号 |
| **Single** | 采集一次就停 | 持续等待 | 一次性事件 |

> **口诀：** Auto 看得到在跑，Normal 看不到在等。
> 
> **经典困惑解答：**
> - "波形在跑" → 触发电平设错了，调 Level 到波形中间
> - "屏幕不动了" → Normal 模式下触发条件没满足，先切 Auto 确认信号还在

---

## 5. 高级触发类型

| 类型 | 原理 | 应用 |
|:---|:---|:---|
| **脉宽触发** | 脉冲宽度满足条件时触发（>、<、= 某值） | 捕获异常宽/窄脉冲 |
| **欠幅触发** | 信号"没跳到位"时触发 | 总线信号未达到有效电平 |
| **建立/保持时间触发** | 时序违例时触发 | FPGA/ASIC 时序调试 |
| **超时触发** | 信号超时未变化时触发 | 通信超时、中断丢失 |
| **串行总线触发** | 按协议内容触发（I2C/SPI/UART/CAN/USB） | 协议调试 |

---

## 6. 触发释抑（Trigger Holdoff）

触发后**拒绝再次触发**的一段时间。用于信号有多个相似边沿但只想在特定边沿触发的场景。

```
无释抑：每个上升沿都触发 → 波形不稳定
有释抑：只在第一个上升沿触发 → 波形稳定
```

**设置方法：** Trigger Menu → Holdoff → 调整时间

---

## 7. 触发故障排查

| 现象 | 可能原因 | 解决 |
|:---|:---|:---|
| 波形左右漂移 | 触发电平不在信号范围内 | 调 Level 到波形中间 |
| 有时稳定有时漂 | 触发电平在噪声附近 | 微调 Level 避开噪声 |
| 屏幕不动（Normal） | 触发条件不满足 | 切 Auto 确认信号存在 |
| 波形跳动 | 触发沿选错 | 换上升/下降沿 |
| 每次位置不同 | 多个相似边沿 | 设置 Holdoff |
| 显示 "Trig?" | 触发条件未满足 | 检查源/电平/模式 |
