**电机初始化不是为了让电机动，而是为了保证电机绝对不乱动。**  
真正让电机动，应该发生在 motor_driver_tb6612_cmd(1) 之后，再调用 set_duty()


这个阶段的教学重点是：**速度不是编码器中断里算的，而是在固定周期任务里算的**。中断只负责快速计数，周期任务负责估算速度。这样中断短、系统稳定，后面接 PID 也更舒服。



这个模块的职责：

`输入：target 和 measurement 内部：计算 error、积分、微分 输出：control_output`

它不应该知道：

`电机 PWM 编码器 轮速 MPU6050 HAL API`

也就是说，PID 是纯算法层。

**第一版选位置式 PID**

推荐第一版用位置式 PID，不用增量式 PID。

原因：

`位置式 PID 更容易理解 输出可以直接对应 duty_percent 容易做输出限幅 适合你现在学习阶段`

基本公式：

`error = target - measurement integral += error * dt derivative = (error - last_error) / dt output = kp * error + ki * integral + kd * derivative`

然后做两个保护：

`integral 限幅 output 限幅`
