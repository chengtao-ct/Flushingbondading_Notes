> [!abstract] 题目概述
> **考察方向：** 快慢双指针 + 原地修改（In-place）。
> **嵌入式场景：** 你的单片机 Flash 空间快满了，里面存了一批按时间排序的历史温度数据，但有大量连续重复的读数。你需要在不申请新 RAM 的情况下，把重复项"挤掉"，将有效数据紧凑地排列在数组最前面。
> **硬性要求：** 空间复杂度必须是 `O(1)`。

> [!note] 思考过程
> - **想法**：先明确重复的数，还是找到一个，移动一个，最后清0。
> - **解法**：不让用数组没招了。但是我死都没有想到前后比较这个关键点，草。

## 解法

```c
int removeDuplicates(int* nums, int numsSize) {

   if(numsSize <= 1)

   {

    return numsSize;

   }

    int slow_p = 1;//覆写位置

    for(int fast_p = 1; fast_p < numsSize; fast_p++)

    {

        if(nums[fast_p] !=nums[fast_p-1])

        {

            nums[slow_p]= nums[fast_p];

            slow_p++;

        }

    }

    return slow_p;

}
```

> [!tip] 核心收获
> 最重要的是看你如何理解这几个指针的作用。
