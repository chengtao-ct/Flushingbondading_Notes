- **复习目标：** 双层循环滑动窗口的 C 语言实现。
    
- **挑战点：** 回忆我们刚才说的“边界剪枝”（外层循环只要走到 `len1 - len2` 即可），以及在内层循环比对失败时，如何干净利落地 `break` 跳出。



**思考**：
- 按照逻辑上来说应该是进行两次遍历，needle的头匹配和末尾匹配，而且只需要首次的索引
- 获得遍历长度
- 反正先进行特殊情况的考虑肯定没问题
- 假如匹配上了，外层循环肯定不是从零开始，好的解决了

**问题**：外层终止在哪里


```c
int strStr(char* haystack, char* needle) {
    int n = strlen(haystack);
    int m = strlen(needle);

    // 1. 规矩：要找的串是空的，直接返回 0
    if (m == 0) {
        return 0;
    }

    // 2. 边界：必须是 <=，防止长度一样时不进循环
    for (int i = 0; i <= n - m; i++) {
        
        int j; // 3. 作用域：把 j 的声明提出来，让下面的 if 能认出它
        
        for (j = 0; j < m; j++) {
            // 4. 核心：主串的下标必须是起点 i 加上 偏移量 j
            if (haystack[i + j] != needle[j]) {
                break; // 错一个就立刻放弃这轮
            }
        }
        
        // 如果 j 能完整地走到 m，说明中间没有被 break 踢出来
        if (j == m) {
            return i; // 胜利！返回起始起点 i
        }
    }

    return -1; // 全都滑完了也没找到
}
```
