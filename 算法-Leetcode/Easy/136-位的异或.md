### 核心法则：异或的“自我湮灭”

把异或 (`^`) 想象成一种状态翻转开关。它的规则非常简单：**相同为 0，不同为 1**。 它有三个极其重要的特性，堪称为这道题量身定制：

- **特性 1（自反性）：** 任何数和自己异或，结果永远是 0。（`A ^ A = 0`）。这意味着成对出现的数字，一碰就会“自我湮灭”归零。
    
- **特性 2（恒等律）：** 任何数和 0 异或，结果还是它自己。（`A ^ 0 = A`）。
    
- **特性 3（交换律）：** 异或的顺序不影响结果。（`A ^ B ^ A` 等同于 `A ^ A ^ B`，结果是 `B`）。


```c
int singleNumber(int* nums, int numsSize) {
    int result = 0; // 初始状态为 0，因为 0 异或任何数等于那个数本身
    
    // 单循环：只需遍历一次数组
    for (int i = 0; i < numsSize; i++) {
        // 把每一个数都跟 result 进行异或，并存回 result
        result = result ^ nums[i]; 
    }
    
    // 循环结束后，成对的数都变成了 0，result 里装的就是那个落单的数
    return result;
}
```
