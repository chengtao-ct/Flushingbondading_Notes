# 死区时间

## 核心结论

死区时间是在半桥或 H 桥里，故意让上管和下管都关断一小段时间，避免两个开关同时导通造成直通。

## 工程含义

```mermaid
flowchart LR
    A["上管关断"] --> B["等待死区"]
    B --> C["下管开通"]
```

死区太短，上下管可能因为关断延迟、栅极电荷和驱动传播延迟而同时导通；死区太长，电流会更多地走体二极管或续流路径，增加损耗和反向恢复冲击。

## 软件视角

在 MCU 里，死区常由高级定时器或驱动芯片配置。写 [[TIM定时器基础概念|PWM]] 时要理解：互补输出不是简单把一路信号取反，还要考虑器件真实开关速度和驱动延迟。

## 调试关注

- 用 [[../../../示波器/01-原理与认知/1.1-示波器是什么|示波器]] 同时看上下管栅极波形。
- 看开关节点是否有异常尖峰或体二极管导通过久。
- 从低压限流开始调试，不要第一次就上满母线和大电流。
