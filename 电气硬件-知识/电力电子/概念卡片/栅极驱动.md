# 栅极驱动

## 核心结论

栅极驱动不是“给一个高电平”这么简单，而是在规定时间内给栅极电容充放电，并控制开关速度、误导通、隔离和故障关断。

## 工程判断

| 问题 | 为什么重要 |
|---|---|
| 驱动电压够不够 | 决定 [[MOSFET]] 是否真正低阻导通，[[IGBT]] 是否按规格工作 |
| 峰值电流够不够 | 决定栅极电荷 $Q_g$ 能多快搬运 |
| 栅极电阻多大 | 折中 [[开关损耗]]、过冲、振铃和 EMI |
| 是否需要隔离 | 高边、市电侧、逆变桥通常不能直接共地驱动 |
| 是否有故障关断 | 短路、退饱和、欠压时要硬件级动作 |

## 常见措施

- 栅极串联电阻调节边沿。
- 栅源下拉防止上电悬空误导通。
- 米勒钳位或负压关断抑制高 $dv/dt$ 误开通。
- 半桥/H 桥中设置 [[死区时间]]。
- IGBT 场景配合 DESAT、软关断和隔离驱动。
