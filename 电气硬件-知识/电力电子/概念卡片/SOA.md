# SOA

## 核心结论

SOA 是 Safe Operating Area，中文常叫安全工作区。它告诉你器件在某个电压、电流、时间和温度组合下是否还能安全工作。

## 为什么软件工程师也要看

代码里的限流、启动斜坡、刹车策略、短路保护延迟，都会改变器件承受高电压和高电流的时间。只看“最大电压”和“最大电流”是危险的，因为这两个最大值通常不能同时长期成立。

## 工程判断

| 场景 | SOA 关注点 |
|---|---|
| 电机启动 | 峰值电流持续多久 |
| 堵转 | 高电流是否超过脉冲 SOA |
| 线性调光/软启动 | 器件是否长时间承受高压高流 |
| 短路保护 | 硬件关断是否快于器件损坏时间 |
| 感性关断 | 雪崩或钳位能量是否可承受 |
