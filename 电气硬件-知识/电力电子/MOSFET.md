# MOSFET

> [!abstract] 核心本质
> MOSFET 是电压驱动型功率器件。MCU 或驱动芯片通过控制栅源电压 $V_{GS}$ 改变漏源之间的导通状态，特别适合低压、高速、高频开关场景。

## 一句话结论

MOSFET 最适合嵌入式里常见的低压电机、LED、电磁阀和 DC-DC 控制；真正的难点不是“能不能导通”，而是栅极驱动、续流路径、开关尖峰和散热。

## 结构与控制方式

功率 MOSFET 最常见的是垂直结构。电流从漏极到源极垂直流过芯片，栅极通过电场控制沟道是否形成。

| 引脚 | 英文 | 作用 |
|---|---|---|
| G | Gate | 栅极，控制端，近似电容输入 |
| D | Drain | 漏极，主电流端 |
| S | Source | 源极，主电流端和栅极参考点 |

MOSFET 的输入端近似电容，稳态几乎不耗直流电流。但开关瞬间必须给栅极电容充放电，所以高频时仍然需要栅极驱动能力。

## 关键参数

| 参数 | 含义 | 嵌入式关注点 |
|---|---|---|
| $V_{DS}$ | 漏源耐压 | 要覆盖电源、尖峰和裕量 |
| $I_D$ | 连续漏极电流 | 必须结合温升和 PCB 散热看 |
| $R_{DS(on)}$ | 导通电阻 | 决定导通损耗 $P \approx I^2R$ |
| $V_{GS(th)}$ | 栅源阈值电压 | 只是“刚开始导通”，不是完全导通电压 |
| $Q_g$ | 栅极总电荷 | 决定驱动电流和开关速度 |
| $C_{iss}$、$C_{oss}$、$C_{rss}$ | 寄生电容 | 影响开关波形、米勒效应和振铃 |
| $E_{AS}$ | 单脉冲雪崩能量 | 感性负载异常时的抗冲击能力 |

> [!warning] 避坑指南
> $V_{GS(th)}$ 不是 MCU 能否直接驱动 MOSFET 的判断标准。它通常只表示几百微安电流下刚刚形成沟道。要看指定 $V_{GS}$ 下的 $R_{DS(on)}$，例如 2.5V、4.5V 或 10V。

## 静态特性

MOSFET 作为开关时主要工作在两个状态：

| 状态 | 条件 | 表现 |
|---|---|---|
| 截止 | $V_{GS}$ 低于有效驱动电压 | 漏源近似断开 |
| 导通 | $V_{GS}$ 足够高 | 漏源表现为小电阻 $R_{DS(on)}$ |

导通损耗近似：

$$
P_{cond} = I_D^2 \times R_{DS(on)}
$$

温度升高会让 $R_{DS(on)}$ 增大，所以 MOSFET 发热后会更热，散热设计必须留裕量。

## 动态特性与米勒平台

MOSFET 开关过程本质是栅极电荷搬运。驱动器给栅极充电时，$V_{GS}$ 并不是线性一路上升，其中会出现米勒平台。

```mermaid
flowchart LR
    A["栅极开始充电"] --> B["达到阈值，电流开始上升"]
    B --> C["米勒平台：$V_{DS}$ 快速下降"]
    C --> D["栅极继续升高，完全导通"]
```

米勒平台期间，驱动电流主要用来改变 $V_{DS}$，这段时间直接影响 [[开关损耗]]。驱动太弱会让开关慢、发热大；驱动太强会让尖峰、振铃和 EMI 变严重。

## 体二极管与续流

功率 MOSFET 内部天然带有体二极管。它能提供续流路径，但体二极管通常不是理想二极管：

- 正向压降可能较高。
- 反向恢复可能较差。
- 在同步整流或桥式驱动中会影响死区设计。

> [!note] 工程启示
> 半桥或 H 桥中，死区太短会直通，死区太长会让体二极管导通时间变长，增加损耗和反向恢复冲击。

## 嵌入式典型接法

### 低边 N 沟道开关

最适合 MCU 入门：负载接电源正端，MOSFET 接地侧。MCU 通过栅极电阻驱动 MOSFET，源极接系统地。

注意点：

- 栅极串联小电阻抑制振铃。
- 栅源并联下拉电阻，防止上电悬空误导通。
- 感性负载要有续流、TVS 或吸收回路。
- 大电流回路和 MCU 地线要处理好回流路径。

### 高边开关

高边 N 沟道 MOSFET 需要栅极电压高于源极，通常要自举驱动或专用高边驱动芯片。若只是简单电源开关，也可以用 P 沟道 MOSFET，但导通电阻和成本通常不如 N 沟道。

## 选型流程

1. 确认最大 $V_{DS}$，包含电源波动和尖峰。
2. 确认工作电流、峰值电流和热条件。
3. 在实际驱动电压下查 $R_{DS(on)}$。
4. 根据 PWM 频率和驱动能力查 $Q_g$。
5. 检查封装热阻、SOA、雪崩能量和体二极管参数。

## 常见误区

- 用 3.3V GPIO 驱动一个只在 10V 下标称低导通电阻的 MOSFET。
- 只看 $I_D$，不看温升曲线和 PCB 铜皮散热。
- 忽略栅极下拉，导致上电瞬间误导通。
- H 桥没有死区，造成上下管直通。
- 感性负载没有吸收回路，导致 $V_{DS}$ 尖峰击穿。
