👉 **1. 一句话总结 (Core Purpose)**  
memset 是一个**内存初始化函数**。它的核心作用是：将一块连续的内存空间（比如一个数组、一个结构体），**全部快速地填满某一个特定的值**。最常见的用法就是用来“清零”。

---

### 🔍 2. 逐层 C 语言语法剖析 (Code Breakdown)

这个函数定义在标准库 <string.h> 中（在 RT-Thread 中通常使用自己实现的 rt_memset，用法完全一样）。

它的原型长这样：

codeC

```
void *memset(void *s, int c, size_t n);
```

**参数拆解：**

- **void *s (起点)：** 你要填写的内存块的起始地址。（比如你要清空的数组名，或者结构体指针）。
    
- **int c (填什么)：** 你想要往内存里填入的数据。**注意：虽然它是 int 类型，但 memset 实际上是按“字节（Byte）”来填写的**，所以通常填入 0（清零）或者特定的字符（如 0xFF）。
    
- **size_t n (多长)：** 你要填写多少个字节？（通常用 sizeof 运算符来计算）。
