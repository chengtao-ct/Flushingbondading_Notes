模块 2：核心概念——时域与波形
---
2.1 什么是时域（Time Domain）？
同学，在讲波形之前，我们必须先彻底理解一个根本性的概念：域（Domain）。
在信号分析中，我们有两种主要的"视角"来看待一个信号：
两种视角对比
 	时域（Time Domain）
横轴	时间（s, ms, μs, ns）
纵轴	电压（V, mV, μV）
回答的问题	信号随时间怎么变？
用什么仪器看	示波器
类比	一首歌的波形（声波随时间跳动）
时域的本质
时域就是：以时间为横轴，以电压为纵轴，观察信号的变化过程。
这是最直觉、最符合人类感知的分析方式。想象你在看一张心电图：
电压(mV)
  ↑
  │        ╱╲
  │       ╱  ╲        ╱╲
  │  ╱╲  ╱    ╲      ╱  ╲
  │ ╱  ╲╱      ╲    ╱    ╲
  │╱            ╲──╱      ╲──
  └──────────────────────────→ 时间(s)
  0              0.5        1.0
- 心跳每跳动一次，电压先升高再降低，形成一个完整的"峰"
- 你能看到每次跳动的形状、间隔、高度
- 这就是时域分析的全部意义——看信号"长什么样"
> 工程师经验： 你未来 95% 的调试工作都在时域中进行。频域分析（FFT）是进阶技能，后面模块 9 会涉及。先把时域搞透。

2.2 波形的基本构成要素
任何波形，无论多么复杂，都可以用以下几个基本参数来描述。这五个参数是示波器测量的核心，必须背下来：
五大基本参数
电压(V)
  ↑
  │    ╱╲                    ╱╲
  │   ╱  ╲                  ╱  ╲
  │  ╱    ╲                ╱    ╲
  │ ╱      ╲              ╱      ╲
  │╱        ╲            ╱        ╲
  └──────────╲──────────╱──────────╲──→ 时间
  0           ←─ T ──→              ←─ T ──→
参数	符号	单位
幅度（Amplitude）	A	V（伏特）
峰峰值（Peak-to-Peak）	Vpp	V
周期（Period）	T	s（秒）
频率（Frequency）	f	Hz（赫兹）
相位（Phase）	φ	°（度）或 rad（弧度）
最关键的公式——频率与周期的关系
$$f = \frac{1}{T}$$
频率和周期互为倒数。 这个公式你会在工作中用到无数次，必须形成肌肉记忆：
周期 T	频率 f
1s	1 Hz
1ms	1 kHz
1μs	1 MHz
1ns	1 GHz
> 老师笔记： 工程师日常工作中最常见的速算：
> - 看到 1ms 的周期 → 立刻想到 1kHz
> - 看到 100ns 的周期 → 立刻想到 10MHz
> - 看到 1μs 的周期 → 立刻想到 1MHz
> 
> 做一个小练习：如果示波器屏幕上显示一个完整正弦波占 5 格，Time/div 设置为 2ms/div，那么周期 T = 5 × 2ms = 10ms，频率 f = 1/10ms = 100 Hz。
幅度的深入理解
幅度有几种不同的表示方式，新手很容易搞混：
电压
  ↑    Vp（峰值）
  │    ──→ │
  │        ╱╲
  │       ╱  ╲  ←── Vpp（峰峰值）
  │      ╱    ╲
  │     ╱      ╲
  │────╱────────╲──────────→ 时间
  │              ╲
  │               ╲
  │   Vrms        ╱╲
  │    ──→ │      ╱  ╲
  │        │     ╱    ╲
  └────────────────────
         0V（基线）
幅度类型	符号	含义
峰值	Vp	从零线到波峰的距离
峰峰值	Vpp	从最低谷到最高峰的距离
有效值（均方根值）	Vrms	信号做功能力的等效直流电压
> 为什么 Vrms 重要？
> 因为它代表了信号的实际能量。比如：
> - 中国市电标称"220V"，这个 220V 就是 Vrms
> - 实际峰值 Vp = 220 × √2 ≈ 311V
> - 峰峰值 Vpp ≈ 622V
> - 用示波器测市电时，你会看到波形在 -311V 到 +311V 之间摆动！ 如果不知道这一点，会非常惊恐。
> 
> 安全警告： 新手千万不要用示波器直接测市电！后面模块 10 会详细讲为什么以及安全做法。

2.3 常见波形类型
在电子工程中，你会反复遇到以下几种波形。我现在逐一讲解它们的形状、特征和应用场景。
① 正弦波（Sine Wave）
     ╱╲          ╱╲          ╱╲
    ╱  ╲        ╱  ╲        ╱  ╲
───╱    ╲──────╱    ╲──────╱    ╲───
         ╲    ╱        ╲    ╱
          ╲  ╱          ╲  ╱
           ╲╱            ╲╱
特征：
- 最"自然"的波形，自然界中无处不在
- 数学表达：V(t) = A × sin(2πft + φ)
- 只包含单一频率成分——纯正弦波没有谐波
典型应用：
- 交流市电（50Hz 正弦波）
- 音频信号的基础成分
- 晶体振荡器输出（非常接近纯正弦波）
- 射频（RF）信号
> 老师笔记： 正弦波是信号分析的"基石"。任何复杂的波形都可以分解为多个正弦波的叠加——这就是傅里叶变换（Fourier Transform） 的核心思想。虽然你现在不需要深入理解傅里叶，但要知道：正弦波是一切波形的"原子"。
---
② 方波（Square Wave）
   ┌──────┐      ┌──────┐      ┌──────┐
   │      │      │      │      │      │
───┘      └──────┘      └──────┘      └───
特征：
- 在两个固定电压值之间瞬间跳变
- 理想方波的上升/下降时间为零（现实中不可能）
- 包含奇次谐波（1次、3次、5次、7次……）
典型应用：
- 数字电路的时钟信号（Clock）——这是最常见的方波！
- PWM 信号（脉宽调制）
- 单片机的 GPIO 输出（翻转高低电平）
- 555 定时器输出
> 工程师经验： 方波是你用示波器看得最多的波形。调试单片机、FPGA、数字电路时，你每天都要看时钟信号和数字信号是否正确。
---
③ 三角波（Triangle Wave）
       ╱╲          ╱╲          ╱╲
      ╱  ╲        ╱  ╲        ╱  ╲
     ╱    ╲      ╱    ╲      ╱    ╲
────╱      ╲────╱      ╲────╱      ╲───
            ╲  ╱        ╲  ╱
             ╲╱          ╲╱
特征：
- 线性上升，然后线性下降
- 上升和下降的斜率绝对值相同
- 包含奇次谐波，但幅度比方波衰减更快
典型应用：
- 扫频信号源（用于测试电路的频率响应）
- 模拟比较器的基准信号
- 音频合成器
- ADC（模数转换器）的测试信号
---
④ 锯齿波（Sawtooth Wave）
│        ╱│        ╱│        ╱│
│       ╱ │       ╱ │       ╱ │
│      ╱  │      ╱  │      ╱  │
│     ╱   │     ╱   │     ╱   │
│    ╱    │    ╱    │    ╱    │
────╱     │────╱     │────╱     │
│       ╱│        ╱│        ╱│
│      ╱ │       ╱ │       ╱ │
特征：
- 线性缓慢上升，然后瞬间下降（或者反过来）
- 上升和下降的速率不对称
- 包含所有次谐波（1次、2次、3次……全部）
典型应用：
- CRT 显示器的水平扫描信号（经典的锯齿波应用）
- 开关电源中的斜坡补偿
- 音频合成器（锯齿波音色与方波不同）
- ADC 中的采样-保持电路
---
⑤ 脉冲波（Pulse Wave）
   ┌┐                   ┌┐
   ││                   ││
───┘└───────────────────┘└────────────
   ←─ tw →  ←── T ──→
特征：
- 方波的"特殊版"——高低电平的时间不相等
- 两个关键参数：
  - 脉宽（Pulse Width, tw）：高电平持续的时间
  - 占空比（Duty Cycle）：高电平时间占整个周期的百分比
$$\text{Duty Cycle} = \frac{t_w}{T} \times 100\%$$
典型应用：
- PWM 调光（LED 亮度控制）：占空比越大，LED 越亮
- PWM 调速（电机控制）：占空比越大，电机转速越快
- 超声波测距的发射脉冲
- 通信协议中的起始/停止位
> 老师笔记： 占空比是脉冲波最重要的参数。
> - 占空比 50% 的脉冲波 = 方波
> - 占空比 1% = 极窄的脉冲（常用于雷达、超声波）
> - 占空比 99% = 几乎一直高电平
---
⑥ 噪声与复合波（Noise & Complex Waveforms）
在现实中，你看到的波形往往不是"标准"的——它们可能叠加了噪声、毛刺、振铃等：
理想方波：     实际方波（有振铃和过冲）：
  ┌──┐          ┌─╱╲──┐
  │  │          │╱    ╲│
──┘  └──    ───╱╲      ╲──
              ╱  ╲  ╱╲  ╲
             ╲   ╲╱  ╲  ╱
              ╲       ╲╱
常见的"不完美"波形现象：
现象	英文名	描述
过冲	Overshoot	信号跳过了目标值再回来
下冲	Undershoot	信号跌破了目标值再回来
振铃	Ringing	信号在目标值附近来回振荡
毛刺	Glitch	短暂的异常电压尖峰
抖动	Jitter	信号边沿在时间上的随机偏移
> 工程师经验： 在数字电路调试中，你看到最多的"问题波形"就是振铃和过冲。它们通常不会让电路立刻失败，但会降低信号的噪声裕量（Noise Margin），导致系统在极端条件下（温度变化、电源波动）出现间歇性故障——这种 bug 最难查。

2.4 波形的数学表达（简述）
不需要你成为数学家，但以下基本公式你应该认识：
波形	数学表达式
正弦波	V(t) = A × sin(2πft + φ)
方波	V(t) = A × sgn(sin(2πft))
三角波	V(t) = (2A/π) × arcsin(sin(2πft))
锯齿波	V(t) = 2A × (ft - ⌊ft + 0.5⌋)
其中：
- A = 振幅
- f = 频率
- t = 时间
- φ = 初相位
- sgn = 符号函数（正为1，负为-1）
> 老师笔记： 这些公式不需要背，但你需要理解每个参数的物理意义。在示波器上调节旋钮时，你实际上就是在改变这些参数。
