模块 6：示波器探头（Probe）
---
6.1 探头为什么重要？
同学，很多新手花大价钱买了好的示波器，却配了一根几十块钱的廉价探头——这就像给一辆跑车装了自行车轮胎。
探头是示波器和被测电路之间的"桥梁"。 信号的真实面貌取决于这座桥的质量：
被测电路  ──── 探头 ──── 示波器 ──── 你的眼睛
   │                      │
   │  探头会：            │  示波器会：
   │  · 衰减信号          │  · 采样信号
   │  · 滤除高频          │  · 存储数据
   │  · 引入噪声          │  · 显示波形
   │  · 改变电路行为      │
   │                      │
   └── 探头质量直接决定你看到的波形是否"真实" ──┘
> 工程师经验： 在实际调试中，大约 30% 的"信号异常"其实是探头本身引入的假象。 学会正确使用探头，能帮你避免大量误判。
---
6.2 探头的等效电路模型
要理解探头，必须先理解它的电路模型。探头不是一根简单的导线，它是一个复杂的电路：
被测电路                    示波器
   │                         │
   │    ┌─── R_probe ────┐   │
   ├───┤                 ├───┤─── R_scope (1MΩ)
   │    │   C_probe      │   │
   │    ├───||───────────┤   ├───||── C_scope (~15pF)
   │    │                │   │
   │    └── L_cable ─────┘   │
   │      (寄生电感)          │
   │                         │
  GND──┤─────────────────────┤─── GND
       │    (地线电感)        │
关键参数：
参数	典型值	影响
R_probe（探头电阻）	10MΩ（10X） / 1MΩ（1X）	对电路的负载效应
C_probe（探头电容）	8~15pF（10X）	高频信号衰减、边沿变缓
C_scope（示波器输入电容）	~15pF	与探头电容一起影响测量
L_cable（电缆寄生电感）	~200nH/m	高频谐振、振铃
地线电感	取决于地线长度	接地不良时产生振铃
探头的负载效应
探头接到电路上后，改变了电路原本的行为：
不接探头时的信号：           接上探头后的信号：
  ┌──┐                      ┌──╱╲──┐
  │  │                      │╱     ╲│
  │  │                      │       │
  └──┘                      └───────┘
  理想方波                   边沿变缓了！
                           因为探头的电容"吃掉"了高频分量
探头电容越大，对高频信号的干扰越大。
探头类型	输入电容	对电路的影响
1X 无源探头	~100pF	很大！严重改变高频电路行为
10X 无源探头	815pF	中等，大多数场景可接受
有源探头	12pF	很小，适合高速信号
差分探头	13pF	很小，且可以测差分信号

6.3 无源探头（Passive Probe）——最常用
3.1 结构拆解
探针尖端                    BNC 接口
   │                          │
   ▼                          ▼
  ─┤├─  ← 尖端              ──┤├──  ← BNC 头
   │                          │
   │   ┌──── 同轴电缆 ────┐    │
   │   │                   │    │
   │   │  内部有：          │    │
   │   │  · 信号线（芯线）  │    │
   │   │  · 9MΩ 补偿电阻  │    │
   │   │  · 补偿电容       │    │
   │   │  · 屏蔽层（地）    │    │
   │   │                   │    │
   │   └───────────────────┘    │
   │                          │
  ┌┴┐                        │
  │ │  ← 接地夹（鳄鱼夹）      │
  └─┘                        │
                              │
  ┌───┐                      │
  │ × │  ← 1X/10X 开关       │
  └───┘                      │
3.2 10X 档位的分压原理
10X 探头内部有一个 9MΩ 的电阻与示波器输入端的 1MΩ 电阻形成分压器：
被测信号
   │
   ├── R_probe = 9MΩ ──────┬─── R_scope = 1MΩ ───→ 示波器 ADC
   │                       │
   ├── C_probe ──────────||─┤── C_scope ──||──→ 示波器 ADC
   │                       │
  GND ─────────────────────┘
分压比 = R_scope / (R_probe + R_scope)
       = 1MΩ / (9MΩ + 1MΩ)
       = 1/10
所以示波器上显示的电压 = 实际信号电压 × 1/10
→ 示波器自动 × 10 显示正确值
3.3 1X 档位的工作方式
拨到 1X 时，探头内部的 9MΩ 电阻被旁路，信号直接进入示波器：
1X 模式：
被测信号 ───────────────────────→ 示波器 ADC（直连）
          没有 9MΩ 分压电阻
          信号不衰减
1X 的严重问题：
输入电容对比：
10X 模式：  电路 ── 9MΩ ── [C ≈ 8~15pF] ── 示波器
                         ↑
                      电容较小
1X 模式：   电路 ── [C ≈ 100pF] ── 示波器
                   ↑
                电容非常大！
100pF 的电容对高频信号的旁路效应：
  高频信号 → 被 100pF 电容"短路"到地 → 信号幅度下降、边沿变缓
> 老师笔记： 1X 档位的 100pF 输入电容是什么概念？一根 10cm 长的 PCB 走线的寄生电容大约才 2~5pF。1X 探头相当于在你电路上额外并联了一个 100pF 的电容，这在高速电路中是灾难性的。
>
> 原则：能用 10X 就用 10X，1X 只用于低频、低幅度信号。
3.4 10X vs 1X 完整对比
对比项	10X	1X
衰减比	10:1（信号缩小 10 倍）	1:1（信号不衰减）
输入电阻	10MΩ	1MΩ
输入电容	8~15 pF	~100 pF
最大测量电压	~300Vpp（低压探头）	~30Vpp
带宽	全带宽（取决于探头额定值）	610 MHz
对电路影响	小	大
最小可测信号	~10mV（受示波器灵敏度限制）	~1mV
适用场景	绝大多数场景	极低幅度、低频信号

6.4 探头补偿（Probe Compensation）——必须掌握！
4.1 为什么需要补偿？
探头和示波器是两个独立的设备，它们的电容参数不可能完全匹配。如果不做补偿，高频信号会被错误地放大或衰减。
探头补偿就是调节探头内部的补偿电容，使探头的高频响应与示波器匹配。
4.2 补偿的操作步骤
每换一根探头或换一台示波器，都必须做一次补偿！
步骤 1：找到示波器上的校准信号输出端
┌──────────────────────────────────────┐
│  示波器面板上有一个小接口，标着：      │
│  "CAL" 或 "~1kHz 3Vpp" 或类似标记   │
│                                       │
│  这个接口输出一个标准的 1kHz 方波      │
│  用于探头补偿校准                      │
└──────────────────────────────────────┘
步骤 2：将探头钩住校准信号端子
  ┌──────────┐
  │  示波器   │
  │           │
  │  CAL ────┤├── 探头钩子钩这里
  │           │
  │  GND ────┤├── 接地夹夹这里
  └──────────┘
步骤 3：示波器设置
  V/div → 约 1V/div（能看到完整方波）
  Time/div → 约 1ms/div（能看到几个周期）
  触发 → CH1 上升沿，Level 约 1.5V
步骤 4：观察屏幕上的方波形状
步骤 5：用小螺丝刀调节探头补偿电容
  ┌───┐
  │   │  ← 探头 BNC 端附近有一个小孔
  │   │     里面有一个可调电容
  │ × │  ← 用绝缘小螺丝刀伸进去旋转
  └───┘
4.3 三种补偿状态
这是你必须学会判断的三种波形：
欠补偿（Under-compensated）
  ┌──╲    ╱──╲    ╱──╲
  │    ╲╱    ╲╱    ╲╱
  │
──┘
  ↑
  方波的高频分量被衰减了
  角变圆了，边沿不够陡
原因： 探头补偿电容太小
解决： 顺时针旋转补偿电容，增大电容值
---
过补偿（Over-compensated）
  ┌╱╲      ╱╲      ╱╲
  │  ╲    ╱  ╲    ╱  ╲
  │   ╲  ╱    ╲  ╱    ╲
──┘    ╲╱      ╲╱
        ↑
  方波的高频分量被过度放大了
  出现了"尖角"甚至"过冲"
原因： 探头补偿电容太大
解决： 逆时针旋转补偿电容，减小电容值
---
正确补偿（Correctly compensated）
  ┌──┐     ┌──┐     ┌──┐
  │  │     │  │     │  │
  │  │     │  │     │  │
──┘  └─────┘  └─────┘  └──
  ↑
  方波的边沿是平的（不是圆的也不是尖的）
  顶部和底部都是平坦的
这是你要调到的目标状态！
4.4 补偿调节口诀
┌─────────────────────────────────────────────┐
│                                               │
│   角变圆了 → 欠补偿 → 电容太小 → 顺时针转     │
│   角变尖了 → 过补偿 → 电容太大 → 逆时针转     │
│   角是平的 → 补偿正确 → 搞定！               │
│                                               │
└─────────────────────────────────────────────┘
4.5 补偿的频率响应原理（进阶理解）
频率响应曲线：
增益
  ↑
  │        ─── 正确补偿（平坦）
  │       ╱
  │  ╱───╱ ─── 欠补偿（高频下降）
  │╱    ╱
  │     ╱   ╲── 过补偿（高频上升）
  │    ╱      ╲
  └──────────────────→ 频率
- 欠补偿： 高频信号被衰减 → 方波边沿变圆
- 过补偿： 高频信号被放大 → 方波出现尖峰
- 正确补偿： 全频段响应平坦 → 方波形状正确
> 工程师经验： 我见过太多新手因为没做探头补偿就测信号，测出来的方波有"尖角"或"圆角"，以为是信号本身的问题，折腾半天才发现是探头没补偿。每次使用示波器前，花 30 秒做一次探头补偿，能避免你浪费几个小时。

6.5 接地问题——探头使用中最容易被忽视的大问题
5.1 探头接地线不是"可有可无"的
很多新手觉得接地夹"只是个参考"，夹不夹都行——这是完全错误的。
不接地的后果：
探针尖端接触被测点，但接地夹悬空：
  → 示波器通过电源线的地线（大地）找到回路
  → 这个回路可能非常长，包含大量的寄生电感
  → 测量结果完全不可信
类比：
  你量电压却不接万用表的 COM 端 → 读数为 0 或乱跳
  同理，探头不接地 → 波形完全不对
5.2 接地线长度的影响（极其重要！）
接地线越长，引入的寄生电感越大，测量高频信号时产生的振铃越严重。
接地线短（5cm）：               接地线长（20cm）：
  ┌──┐                         ┌──╱╲──┐
  │  │                         │╱     ╲│
  │  │                         │  ╱╲  ╲│
──┘  └──                       │ ╱  ╲  ╲│
  干净的方波                   ╲╱    ╲  ╲│
                               └──────╲─╲│
  ↑                                    ╲╲│
  振铃几乎看不见                       大量振铃！
                                       这不是信号的问题，
                                       是接地线太长引入的！
寄生电感公式：
$$L \approx 1\text{nH/mm} \text{（近似值）}$$
接地线长度	寄生电感	对 100MHz 信号的影响
3 cm（推荐）	~30 nH	轻微振铃
10 cm（标准鳄鱼夹线）	~100 nH	明显振铃
20 cm（拉长了）	~200 nH	严重振铃，波形失真
50 cm	~500 nH	波形完全不可信
5.3 如何改善接地？
方法	操作	效果
尽量缩短接地线	把鳄鱼夹夹在最近的 GND 点	最有效
使用接地弹簧	有些探头配了短弹簧替代鳄鱼夹线	高频测量必备
使用 PCB 接地弹簧	将弹簧焊在 PCB GND 上	专业做法
使用接地平面	用铜箔把探头地接到大面积地平面	最高频测量
推荐的接地方式：
方式1：标准鳄鱼夹（低频 OK）
  探针尖端 ─── 被测点
  接地夹 ──── 最近的 GND（尽量短）
方式2：接地弹簧（高频推荐）
  探针尖端 ─── 被测点
  接地弹簧 ── 紧邻的 GND 焊盘
  （弹簧长度只有 1~2cm）
方式3：BNC 接地适配器（最高频）
  探头通过适配器直接旋入 PCB 的 SMA 接口
  几乎零接地电感

6.6 有源探头（Active Probe）
6.1 什么是有源探头？
有源探头在探头内部集成了一个放大器电路（需要供电），而不是像无源探头那样只有无源元件（电阻、电容）。
无源探头：                    有源探头：
  被测点 ── R/C ──────── 示波器   被测点 ── [放大器] ──── 示波器
                               ↑
                            需要供电（通常从示波器取电）
6.2 有源探头的优势
参数	无源探头 10X	有源探头
输入电容	8~15 pF	0.5~2 pF
带宽	100~500 MHz	1~6 GHz
输入电阻	10 MΩ	1 MΩ
对电路负载	中等	极小
价格	几百元	几千到几万元
需要供电	否	是
有源探头适合：
- 带宽 > 500MHz 的高速信号
- 对寄生电容极其敏感的电路
- DDR、PCIe、USB 3.0 等高速总线
- 射频（RF）电路
> 工程师经验： 如果你用的示波器带宽 ≤ 200MHz，无源探头足够了。只有当你需要测量 > 500MHz 的信号时，才需要考虑有源探头。

6.7 差分探头（Differential Probe）
7.1 什么是差分信号？为什么需要差分探头？
差分信号：两个信号线之间的电压差携带信息。
单端信号（Single-ended）：
  信号线 ──→ 电压相对于 GND
  GND    ──→ 参考点
差分信号（Differential）：
  信号+  ──→  V+    ──┐
                      ├──→ 差分电压 = V+ - V-
  信号-  ──→  V-    ──┘
常见差分信号：
- USB（D+/D-）
- CAN 总线（CAN_H/CAN_L）
- RS-485（A/B）
- LVDS
- 以太网
- DDR 的 DQS/DQ
7.2 为什么不能用两个普通探头测差分信号？
用两个普通探头（CH1 和 CH2）测差分信号：
  信号+ ── CH1 探头 ──→ 示波器 CH1
                          │
  GND  ── CH1 接地夹 ────┤── 示波器内部 GND ──── 公共地！
                          │
  信号- ── CH2 探头 ──→ 示波器 CH2
                          │
  GND  ── CH2 接地夹 ────┘
问题：两个接地夹都接 GND，但信号+ 和信号- 都不是以 GND 为参考的！
     你实际测到的是 V+ 相对于 GND 和 V- 相对于 GND 的值
     然后用 Math 功能做 CH1 - CH2 → 理论上可以得到差分值
     但这有两个严重问题：
  ① 两个探头的特性不完全一致 → 误差
  ② 如果差分信号的共模电压很高 → 可能超过探头耐压！
7.3 差分探头的工作原理
差分探头：
  信号+ ──→┐
            ├──→ [差分放大器] ──→ 单端输出 ──→ 示波器 CH1
  信号- ──→┘
  差分放大器只放大 V+ - V- 的差值
  自动抑制共模电压（V+ + V-）/ 2
  内部有高压隔离 → 可以测高压差分信号
7.4 差分探头的关键参数
参数	含义	典型值
差分电压范围	能测量的最大差分电压	±几十V 到 ±几千V
共模抑制比（CMRR）	抑制共模信号的能力	60dB~80dB
带宽	能测量的信号频率	50MHz~1GHz+
共模电压范围	信号+和信号-的平均电压范围	±几十V 到 ±上千V
> 工程师经验： 测量电机驱动、开关电源、逆变器等有高压浮地的电路时，差分探头是必需品。用普通探头测这类电路，轻则测量错误，重则烧毁示波器（因为地线短路了高压节点）。

6.8 电流探头（Current Probe）
8.1 示波器能直接测电流吗？
不能。 示波器本质上是一个电压测量仪器。要测电流，需要将电流转换成电压。
8.2 两种测电流的方法
方法一：串联采样电阻
  电路 ──[R_sense]── 负载
           │
           └── 示波器探头（测 R 两端电压）
           
  电流 = V_R / R_sense
优点： 便宜、简单
缺点： 需要断开电路、引入额外电阻、可能影响电路工作
方法二：电流探头（推荐）
  电路导线 ────────
          ║
         ╔╩╗  ← 电流探头（钳形）
         ╚═╝     不需要断开电路！
          │
          └──→ 示波器（输出正比于电流的电压）
原理： 基于法拉第电磁感应定律（交流探头）或霍尔效应（直流探头）。
类型	原理	能测	典型带宽
交流电流探头	互感器（变压器原理）	只能测交流	1kHz~100MHz
交直流电流探头	霍尔效应 + 互感器	交流 + 直流	DC~100MHz

6.9 探头使用常见错误
错误	后果	正确做法
不做探头补偿	方波边沿失真	每次使用前用 CAL 信号补偿
接地夹不接	波形完全不对	必须接 GND
接地线太长	高频信号出现振铃	尽量缩短接地线
忘记设探头衰减比	电压读数差 10 倍	10X 探头 → 示波器设 10X
用 1X 测高频信号	带宽不足，边沿变缓	高频信号用 10X
两个探头地接不同地	可能造成短路	所有探头共地
用普通探头测高压差分	可能烧毁示波器	用差分探头
探头尖端接触不良	波形不稳定、噪声大	确认钩紧或用弹簧接地

6.10 模块 6 小结
知识点	核心内容
探头不等于导线	探头有电阻、电容、电感，会改变电路行为
10X vs 1X	默认用 10X（负载小、带宽高、耐压高）
探头补偿	用 CAL 信号调节，角圆→欠补偿，角尖→过补偿，角平→正确
接地线长度	越短越好，长了会引入振铃
有源探头	内置放大器，电容极小，适合 > 500MHz 信号
差分探头	测差分信号和高压浮地电路的必需品
电流探头	钳形测量，不需要断开电路
> 老师寄语： 探头是示波器使用中被低估的环节。很多工程师用了一辈子示波器，却从没认真学过探头补偿和接地技巧。好的探头使用习惯能让你少走很多弯路。 下一个模块我们会讲示波器的关键设置——垂直系统和水平系统的深入操作。
