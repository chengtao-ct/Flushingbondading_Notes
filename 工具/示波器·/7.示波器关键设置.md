模块 7：示波器关键设置
---
7.1 本模块要解决什么问题？
同学，前 6 个模块你学到了理论和基本操作。但当你真正坐到示波器前面，面对一台有几十个旋钮和上百个菜单的仪器时，你需要的不是理论，而是——每个旋钮转了会怎样，每个菜单选项选了会怎样。
本模块的目标：让你面对示波器面板上的每一组控制键，都知道该干什么。
---
7.2 垂直系统（Vertical System）——控制信号的"高低"
垂直系统控制的是波形的电压轴（Y 轴）。每条通道（CH1、CH2、CH3、CH4）都有自己独立的垂直控制。
7.2.1 V/div（伏特每格）——垂直灵敏度
物理位置： 垂直控制区，标有 Scale 或 V/div 的旋钮，每个通道一个。
功能： 控制屏幕上每一格代表多少电压。
V/div = 1V 时：              V/div = 500mV 时：              V/div = 5V 时：
  ↑ 5V                       ↑ 5V                          ↑ 5V
  │ ┌──┐                     │ ┌──┐                        │
  │ │  │                     │ │  │                        │
  │ │  │  占4格               │ │  │  占8格                 │ │  占1格
  │ │  │                     │ │  │                        │ │
──┘ └──┘                    ─┘ └──┘                       ─┘
  0V                        0V                            0V
  
  3.3V方波清晰可见            波形被拉伸，看更多细节          波形被压缩成一条线
旋钮步进序列：1 - 2 - 5 序列
旋转方向和步进：
逆时针（波形变大，看更小信号）：
  10V → 5V → 2V → 1V → 500mV → 200mV → 100mV → 50mV → 20mV → 10mV → 5mV → 2mV → 1mV
顺时针（波形变小，看更大信号）：
  1mV → 2mV → 5mV → 10mV → 20mV → 50mV → 100mV → 200mV → 500mV → 1V → 2V → 5V → 10V
> 工程师经验： 波形占屏幕3~6 格最理想。
> - 太小（< 2 格）→ 调小 V/div（逆时针）
> - 太大超出屏幕 → 调大 V/div（顺时针）
> - 快速技巧：按一下 Scale 旋钮，在粗调（Coarse）和细调（Fine）之间切换。粗调按 1-2-5 序列跳，细调可以连续微调。
7.2.2 V/div 的选择指南
信号幅度	推荐 V/div
3.3V 逻辑	1V/div
5V 逻辑	1V/div 或 2V/div
12V 电源	5V/div
220V 市电	100V/div
100mV 传感器信号	20mV/div
1V 正弦波	200mV/div
7.2.3 Vertical Position（垂直位置）
物理位置： Scale 旋钮旁边，标有 Position 的旋钮。
功能： 上下移动波形，不改变电压比例。
Position 上旋：              Position 下旋：
  ↑  3V ───┐               ↑  3V ───┐
  │        │               │        │
  │  1.5V ─┼────          │  1.5V ─┤
  │        │               │        │
  │  0V ───┘               │  0V ───┼────
  └────────→                └────────→
  波形整体上移              波形整体下移
使用场景：
- 两个通道的波形重叠在一起 → 用 Position 分开
- 想精确读取波形上某一点的电压 → 移到网格线上
- 想对齐两个通道的参考零点
7.2.4 耦合方式（Coupling）——回顾与深入
在模块 3.5 中你已经学过 DC/AC/GND 三种耦合。这里从操作角度深入。
操作路径：
按 CH1 按钮 → 屏幕底部弹出菜单：
┌──────────┬──────────┬──────────┬──────────┬──────────┐
│  耦合     │  带宽限制 │  探头    │  反相    │  挡位调节  │
│  DC      │  全带宽   │  10X    │  关      │  粗调     │
│  AC      │  20MHz   │  1X     │  开      │  细调     │
│  GND     │          │         │          │          │
└──────────┴──────────┴──────────┴──────────┴──────────┘
  ↑ 用前面板软键选择
三种耦合的实际操作场景：
DC 耦合（默认推荐）
看到：信号的全部内容（直流 + 交流）
适用：几乎所有日常调试
典型画面（3.3V 方波）：
  3.3V ───┐     ┌────
           │     │
  0V  ─────┘     └────
  
  你能读到：高电平 3.3V，低电平 0V，Vpp = 3.3V
AC 耦合
看到：去掉了直流偏移的交流成分
适用：观察电源纹波、小信号叠加在大直流上
典型画面（12V 电源上的 50mV 纹波）：
DC 耦合时：                AC 耦合时：
  12V ─≈≈≈≈≈≈≈≈≈≈≈≈≈      +25mV ──┐  ┌──┐  ┌──
  （纹波完全看不清）                 │  │  │  │
                                    └──┘  └──┘──
  -25mV
  纹波被放大到满屏，清晰可见！
GND 耦合
看到：一条 0V 参考线（断开输入）
适用：找到 0V 基线位置
典型画面：
  ──────────────────────  ← 一条水平线在 0V
  （电压 = 0V）
GND 耦合的实际用途：
场景：你想知道波形的 0V 基线在哪里
步骤 1：设为 GND 耦合 → 屏幕上出现一条水平线（0V 基线）
步骤 2：用 Position 旋钮把这条线移到一个网格线上（比如正中间）
步骤 3：切回 DC 耦合 → 现在波形以那个网格线为 0V 参考
  GND 耦合时：              切回 DC 后：
  ────────────── 0V        ────────────── 0V
  （把基线对齐网格线）         ╱╲     ╱╲
                            ╱  ╲   ╱  ╲
                           ╱    ╲ ╱    ╲
  ↑ 这条线就是 0V           现在能清楚看到正负半周
7.2.5 带宽限制（Bandwidth Limit）
这是很多人不知道但非常有用的功能。
操作： CH1 菜单 → 带宽限制 → 全带宽 / 20MHz
设置	含义	效果
全带宽	不限制（使用示波器全部带宽）	能看到高频成分，但噪声也全进来
20MHz	低通滤波，截止频率 20MHz	高频噪声被滤除，波形更干净
全带宽：                      20MHz 带宽限制：
  ┌──╱╲──┐                  ┌──┐   ┌──┐
  │╱    ╲│                  │  │   │  │
  │ ╱╲╱╲ │                  │  │   │  │
──┘      └──              ──┘  └───┘  └──
  有高频噪声和振铃           干净多了！噪声被滤掉了
什么时候用带宽限制？
场景	推荐
看 100MHz 以上的高速信号	全带宽
看音频信号（< 20kHz）	20MHz 限制（更干净）
看 PWM / 电机驱动信号（< 100kHz）	20MHz 限制（滤掉开关噪声）
看电源纹波	20MHz 限制
调试低速数字电路（UART/I2C）	20MHz 限制
> 工程师经验： 很多时候你看到的"信号异常"其实是高频噪声。按一下带宽限制，如果异常消失了，说明那不是信号的问题，而是噪声。这是一个快速判断信号是否干净的技巧。
7.2.6 反相（Invert）
操作： CH1 菜单 → 反相 → 开/关
反相关闭：                   反相开启：
  ┌──┐     ┌──┐             ──┐     ┌──────┐
  │  │     │  │               │     │      │
  │  │     │  │               │     │      │
──┘  └─────┘  └──          ──┘─────┘      └──
  正常方波                    波形上下翻转
使用场景：
- 测量差分信号时，一个通道反相
- 对比正负逻辑信号
- Math 运算中需要取反

7.4 输入阻抗（Input Impedance）—— 1MΩ vs 50Ω
这是一个容易被忽视但极其重要的设置。
7.4.1 两种输入阻抗
阻抗	适用场景
1MΩ（高阻抗）	通用测量，大多数场景
50Ω（低阻抗）	高频/射频测量，匹配传输线阻抗
7.4.2 什么时候用 50Ω？
50Ω 输入阻抗的使用场景：
① 测量高频信号（> 100MHz）时，需要阻抗匹配
  信号源 ── 50Ω同轴电缆 ──→ 示波器 50Ω 输入
            │                   │
            50Ω 特性阻抗         50Ω 输入阻抗
            │                   │
            └── 阻抗匹配，无反射 ──┘
② 测量射频（RF）信号时
③ 使用有源探头时（探头需要 50Ω 终端）
如果用 1MΩ 测高频信号会怎样？
1MΩ 输入测 100MHz 信号：
  信号源 ── 50Ω电缆 ──→ 示波器 1MΩ 输入
                        ↑
                     阻抗不匹配！
                     → 信号在电缆末端反射
                     → 波形上出现振铃、过冲
                     → 测量结果不准确
7.4.3 切换方法
操作路径：
按 CH1 → 菜单中找 "阻抗" 或 "Impedance" → 1MΩ / 50Ω
注意：
  - 不是所有入门级示波器都支持 50Ω 输入
  - 有些示波器 50Ω 只在某些通道上支持
  - 切到 50Ω 时要注意最大输入电压限制（通常 ≤ 5V）
> 工程师经验： 如果你做的是普通数字电路/模拟电路调试（< 50MHz），一直用 1MΩ 就对了。只有在做射频、高速串行总线（> 100MHz）调试时才需要 50Ω。切换到 50Ω 前一定要确认信号电压 ≤ 5V，否则可能烧毁示波器输入电路。

7.5 采样率（Sample Rate）
7.5.1 什么是采样率？
示波器的 ADC（模数转换器）每秒钟对信号采样的次数。
连续信号：    ╱╲      ╱╲
            ╱  ╲    ╱  ╲
           ╱    ╲  ╱    ╲
  ───────╱      ╲╱      ╲──────
采样（每个 × 是一个采样点）：
            ×   ×  ×   ×   ×
           ×  × ×  × ×  × ×
  ───────× ××  ×××  ×××  ××──────
采样率 = 每秒采集多少个 × 点
单位： GSa/s（Giga Samples per second，每秒十亿次采样）或 MSa/s（Mega Samples per second）
7.5.2 采样率与带宽的关系
$$\text{采样率} \geq 4 \sim 5 \times \text{带宽}$$
这是示波器厂商遵循的经验法则：
示波器带宽	最低采样率（实时）
100 MHz	400~500 MSa/s
200 MHz	800 MSa/s ~ 1 GSa/s
350 MHz	1.4~1.75 GSa/s
1 GHz	4~5 GSa/s
7.5.3 采样率不够会怎样？
实际信号（高频正弦波）：
     ╱╲  ╱╲  ╱╲
    ╱  ╲╱  ╲╱  ╲
  ─╱              ╲─
采样率足够时（每个周期 > 10 个点）：
     ×╲×╱╲×╱╲×╱╲×
    ×  ×  ×  ×  ×
  ─×              ×─
  → 还原出的波形接近原始信号 ✓
采样率不足时（每个周期 < 4 个点）：
     ×    ×    ×
      ×      ×
  ─×              ×─
  → 还原出的波形完全失真 ✗
  → 频率看起来更低（混叠！）
这就是混叠（Aliasing）现象——采样率不足时，高频信号看起来变成了低频信号。
实际 50MHz 信号：           采样不足时看起来像：
  ╱╲╱╲╱╲╱╲╱╲╱╲            ╱╲      ╱╲
  真实频率 50MHz            看起来像 5MHz！
  
  → 你会误判信号的频率
  → 非常危险！
7.5.4 如何避免混叠？
方法	操作
调小 Time/div	让示波器在单位时间内采集更多点
检查采样率指示	屏幕上通常会显示当前采样率
打开峰值检测模式	能捕获快速变化的峰值
确认采样率 ≥ 信号频率的 10 倍	经验法则
> 工程师经验： 大多数现代示波器会在屏幕上显示当前的实时采样率。测量时养成看一眼采样率的习惯。如果你看到采样率比信号频率的 10 倍还低，就要警惕混叠问题了。

7.6 存储深度（Record Length / Memory Depth）
7.6.1 什么是存储深度？
示波器能存储的采样点总数。
$$\text{存储深度} = \text{采样率} \times \text{捕获时间（屏幕总时间）}$$
7.6.2 存储深度的实际意义
固定存储深度 = 1M 点：
Time/div = 1μs 时：
  屏幕总时间 = 10μs
  采样率 = 1M / 10μs = 100 GSa/s  ← 很高，波形很精细
Time/div = 1ms 时：
  屏幕总时间 = 10ms
  采样率 = 1M / 10ms = 100 MSa/s   ← 降低了，但还是够用
Time/div = 1s 时：
  屏幕总时间 = 10s
  采样率 = 1M / 10s = 100 kSa/s    ← 非常低！波形粗糙
结论：存储深度越大，在长时间捕获时越能保持高采样率。
存储深度	捕获 10ms 时采样率	捕获 1s 时采样率
10K 点	1 MSa/s	10 kSa/s
100K 点	10 MSa/s	100 kSa/s
1M 点	100 MSa/s	1 MSa/s
10M 点	1 GSa/s	10 MSa/s
7.6.3 什么时候需要大存储深度？
场景	说明
调试长时间通信协议	捕获完整的 I2C/SPI 传输（可能持续几十ms）
捕获间歇性故障	长时间录制，等故障出现
分析电源上电过程	上电可能持续几百ms
解码长数据包	USB/CAN 等协议的数据包可能很长
> 工程师经验： 入门级示波器（如 Rigol DS1054Z）的存储深度通常为 10K~24K 点。这在大多数场景够用，但做协议解码时会捉襟见肘。如果预算允许，选存储深度 ≥ 1M 点的示波器。

7.7 采集模式（Acquire Mode）
操作： 按 Acquire 按钮
7.7.1 三种采集模式
① 采样模式（Sample / Normal）
最基本模式，每个采样周期只取一个点
  实际信号：╱╲  ╱╲  ╱╲
            ×  ×  ×  ← 每个间隔取一个点
  ─────────×──×──×────────
  
  → 正常使用
适用： 大多数日常测量
② 峰值检测模式（Peak Detect）
每个采样周期取最大值和最小值两个点
  实际信号（有窄毛刺）：     采样模式：          峰值检测模式：
  ───┐╱╲────┐╱╲──       ──────────────      ───┐╱╲────┐╱╲──
     │ ×│    │ ×│            毛刺丢失！         │ ×│    │ ×│
     └──┘    └──┘                              └──┘    └──┘
  窄毛刺                   毛刺太窄被漏采了     最大值和最小值都被捕获
适用： 捕获窄脉冲、毛刺、偶发信号
> 工程师经验： 当你怀疑信号上有毛刺但看不到时，切换到峰值检测模式。这是排查间歇性信号问题的利器。注意：峰值检测模式会让波形看起来"粗糙"，但能确保不遗漏任何异常。
③ 平均模式（Average）
对多次采集的波形取平均值，噪声被平均掉
  单次采集（有噪声）：        平均 64 次后：
  ───╱╲──╱╲──╱╲──         ───╱╲──╱╲──╱╲──
     ╲╱  ╲╱  ╲╱                 干净的波形
     噪声很多                    噪声被平均掉了
操作： Acquire → 平均 → 选择次数（4、16、32、64、128、256...）
适用： 消除随机噪声、提高信噪比
注意： 平均模式只能用于重复性信号。一次性事件或随机出现的信号不能用平均模式。
7.7.2 采集模式速查表
模式	优点	缺点
采样	显示真实波形	可能遗漏窄脉冲
峰值检测	不遗漏毛刺	波形看起来粗糙
平均	噪声小，波形干净	只能用于重复信号

7.8 显示设置（Display）
操作： 按 Display 按钮
7.8.1 常用显示选项
选项	功能	推荐设置
类型	普通矢量（连线）/ 点（Dots）	矢量（看起来更连续）
持续（Persistence）	余辉效果	关闭（日常）/ 开启（看信号变化趋势）
波形亮度	调节波形线条亮度	适中（太亮伤眼，太暗看不清）
网格亮度	调节网格线亮度	适中
格式	YT（电压-时间）/ XY（X-Y 模式）	YT（默认）
7.8.2 余辉（Persistence）——一个很有用的功能
余辉关闭：                   余辉开启：
  屏幕只显示当前一帧           多帧波形叠加显示
  ┌──┐                       ┌──┐
  │  │                       │▓▓│  ← 旧波形逐渐变暗
  │  │                       │▒▒│
  └──┘                       └──┘
  每次刷新完全覆盖上一帧       能看到波形的"变化轨迹"
使用场景：
- 观察信号的抖动（Jitter）：开启余辉后，如果边沿有抖动，你会看到边沿位置处有一条"模糊带"
- 观察信号的幅度变化：余辉能显示信号幅度随时间的波动范围
7.8.3 XY 模式
YT 模式（默认）：             XY 模式：
  电压 vs 时间                CH2 电压 vs CH1 电压
  ↑                           ↑
  │  ╱╲                      │    ╱
  │ ╱  ╲                     │   ╱ 
  │╱    ╲                    │  ╱  
  └──────→ 时间              └──────→ CH1 电压
                              
  看信号随时间的变化            看两个信号之间的关系
XY 模式的典型应用：
应用	CH1（X轴）	CH2（Y轴）	看到的图形
李萨如图形	正弦波A	正弦波B	椭圆/圆/8字形（判断频率比和相位差）
器件特性曲线	电压	电流（通过采样电阻转换）	I-V 特性曲线
调制分析	调制信号	载波信号	调制图形

7.9 完整设置速查表
当你坐到示波器前面，按照这个顺序操作：
┌─────────────────────────────────────────────────────┐
│              示波器快速设置流程                        │
│                                                       │
│  ① 开机，等待初始化完成                                │
│                                                       │
│  ② 探头补偿（用 CAL 信号）                             │
│     → 钩 CAL 端子，接地夹夹 CAL 旁的 GND              │
│     → 调 V/div 到 1V，Time/div 到 1ms                │
│     → 观察方波，调补偿电容到"平角"                     │
│                                                       │
│  ③ 连接被测信号                                       │
│     → 探头钩子钩被测点                                 │
│     → 接地夹夹最近的 GND                               │
│     → 确认探头开关在 10X                               │
│                                                       │
│  ④ 通道设置（按 CH1 按钮）                             │
│     → 耦合：DC                                        │
│     → 探头：10X                                       │
│     → 带宽限制：全带宽（或 20MHz 如果信号频率低）       │
│     → 反相：关                                        │
│                                                       │
│  ⑤ 垂直调节                                           │
│     → 旋转 V/div，让波形占 3~6 格                     │
│     → 旋转 Position，让波形在屏幕合适位置              │
│                                                       │
│  ⑥ 水平调节                                           │
│     → 旋转 Time/div，让屏幕显示 2~5 个完整周期         │
│     → 旋转 Position，让触发点在屏幕中间                │
│                                                       │
│  ⑦ 触发设置                                           │
│     → 触发源：选信号所在的通道                          │
│     → 触发沿：上升沿（或下降沿）                       │
│     → 旋转 Level，让触发电平在信号 50% 处              │
│     → 触发模式：Auto                                   │
│                                                       │
│  ⑧ 采集设置（按 Acquire）                              │
│     → 模式：采样（日常） / 峰值检测（查毛刺） / 平均   │
│                                                       │
│  ⑨ 确认波形稳定，开始测量                              │
│     → 按 Measure 设置自动测量参数                      │
│     → 或按 Cursor 用光标手动测量                       │
│                                                       │
└─────────────────────────────────────────────────────┘

7.10 模块 7 小结
知识点	核心内容
V/div	控制垂直缩放，波形占 3~6 格最理想
Position（垂直）	上下移动波形，分开重叠通道
DC/AC/GND 耦合	DC 看全部，AC 看纹波，GND 找零线
带宽限制	20MHz 限制可滤除高频噪声
Time/div	控制水平缩放，显示 2~5 个周期最理想
1MΩ vs 50Ω 输入	通用用 1MΩ，高频射频用 50Ω
采样率	采样率 ≥ 信号频率 × 10，避免混叠
存储深度	越大越好，长捕获时保持高采样率
采集模式	采样（通用）、峰值检测（查毛刺）、平均（去噪声）
余辉	观察信号抖动和变化趋势
