为什么先设 10X？
> 工程师经验：
> 1. 保护示波器：如果你的信号电压超过 ±30Vpp，1X 档位可能损坏示波器输入电路。10X 档位能承受 ±300Vpp。
> 2. 保护被测电路：10X 档位输入阻抗 10MΩ，对电路的"负载效应"更小，测出来的波形更真实。1X 档位只有 1MΩ，可能会拉低高阻抗节点的电压。
> 3. 保留带宽：1X 档位带宽通常只有 6~10MHz，即使你有一台 100MHz 的示波器，用 1X 也只能看到 6MHz 的信号。10X 才能发挥全带宽。
>
> 一般原则：默认用 10X，只有测极低幅度信号（< 10mVpp）时才切到 1X。
示波器端也要对应设置！
这是新手最容易忘的步骤！ 探头拨到 10X 后，示波器必须"知道"你在用 10X，否则测出来的电压值会错 10 倍。
操作方法：
1. 按 CH1 按键（屏幕下方会出现 CH1 菜单）
2. 在菜单中找到 "探头" 或 "Probe" 选项
3. 选择 10X
屏幕下方菜单：
┌──────────┬──────────┬──────────┬──────────┬──────────┐
│  耦合    │  带宽限制 │  探头    │  反相    │  挡位调节  │
│  DC ──→ │  关 ──→  │ 1X  10X │  关 ──→  │  粗调 ──→│
│          │          │  ↑选这个  │          │          │
└──────────┴──────────┴──────────┴──────────┴──────────┘
> 注意： 现在大多数中高端探头支持"智能识别"——探头开关拨到 10X 后，示波器会自动检测到并切换。但如果你的探头比较旧或便宜，可能没有这个功能，就需要手动设置。



三种耦合模式
耦合模式	作用	能看到什么	看不到什么
DC（直流耦合）	信号全部通过	直流 + 交流 全部	什么都能看到
AC（交流耦合）	隔掉直流分量	只有交流变化	看不到直流偏移
GND（接地）	断开输入，接地	只有 0V 基线	看不到任何信号


> 工程师经验：
> DC 耦合让你看到信号的全部信息——包括直流电平和交流变化。PWM 信号最重要的信息之一就是它的高低电平值（0V 和 3.3V），如果用 AC 耦合，你就看不到这两个绝对值了，只能看到一个居中的方波。
>
> 什么时候用 AC 耦合？ 当你想观察信号上的微小纹波或噪声，而直流分量太大把纹波"淹没"了的时候。比如测量 12V 电源上的 50mV 纹波——用 DC 耦合的话，50mV 的纹波在 12V 的波形上几乎看不到；用 AC 耦合，12V 被去掉了，50mV 的纹波被放大到满屏。
>
> 原则：默认 DC，需要看纹波/噪声时切 AC。



注意事项	原因
先断探头，再动板子	防止钩子拉弯引脚
接地夹必须夹紧	松动的地线是噪声和测量错误的头号原因
两个探头共用一个 GND	示波器内部所有通道的地是连通的
每次只测一组	减少出错概率，出问题好定位
钩子钩好后轻拉确认	接触不良会导致读数错误
测完一组就记录	不要等全测完再回忆，记忆会模糊
只有一轮异常先盯单路	不要怀疑整车逻辑，先排除单路问题
