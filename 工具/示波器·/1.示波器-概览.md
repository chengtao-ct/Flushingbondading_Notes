1.2 示波器的定义
示波器（Oscilloscope，简称 Scope） 是一种电子测量仪器，它能够将电信号随时间变化的规律以图形的方式显示在屏幕上。这个图形叫做 波形（Waveform）。
关键要素拆解：
要素	说明
电信号	通常是电压信号（Voltage Signal），也可以是电流信号（通过电流探头转换）
随时间变化	X 轴是时间，Y 轴是电压——这叫 时域分析（Time Domain Analysis）
图形显示	屏幕上画出波形，直观可读
> 老师笔记： "Oscillo" 源自拉丁语 "oscillare"，意思是"摆动"。示波器最初的设计灵感就来自于观察摆的摆动。所以从名字就能理解——它是用来观察"来回变化"的东西的。

1.3 示波器 vs 万用表——详细对比
这是新手最容易混淆的地方，我给你一个完整的对比表：
对比项	万用表（DMM）
显示方式	数字/指针，单一数值
测量维度	静态（某一时刻的值）
能看到什么	电压 3.3V
能发现什么问题	电压是否在正确范围内
采样方式	每秒采样几次到几十次
适用场景	检查电源电压、电阻值、通断测试
价格	几十到几百元
实际工程例子：
假设你在调试一个 LED 闪烁电路，单片机输出 PWM 信号控制 LED 亮度：
- 用万用表测量：显示 1.65V（PWM 占空比 50% 时的平均值）——你只知道"电压大概是 1.65V"
- 用示波器测量：你能看到完整的方波——频率多少、占空比是否精确 50%、上升沿是否够陡、有没有寄生振荡
> 工程师经验： 在实际工作中，*万用表占 20% 的调试时间，示波器占 80%*。当你调试电路出了奇怪的问题，万用表量不出异常时，90% 的情况是示波器能帮你找到原因。

1.4 示波器的基本外观结构
我们先认识一台典型示波器的"长相"。虽然不同品牌型号外观不同，但核心部分都一样：
┌─────────────────────────────────────────────────────┐
│                    ① 显示屏幕                        │
│              （LCD，显示波形和测量值）                  │
│                                                     │
│                                                     │
├─────────────────────────────────────────────────────┤
│  ② 水平控制区        ③ 垂直控制区       ④ 触发控制区  │
│  ┌──────────┐       ┌──────────┐      ┌──────────┐  │
│  │ Time/Div │       │  V/Div   │      │ Trigger  │  │
│  │ (时基)   │       │ (垂直缩放)│      │ Level    │  │
│  └──────────┘       └──────────┘      └──────────┘  │
├─────────────────────────────────────────────────────┤
│  ⑤ 功能按键区          ⑥ 信号输入接口（BNC接口）      │
│  ┌──────────┐       ┌──┐ ┌──┐ ┌──┐ ┌──┐           │
│  │ Measure  │       │CH1│ │CH2│ │CH3│ │CH4│        │
│  │ Cursor   │       └──┘ └──┘ └──┘ └──┘           │
│  │ Math     │                                      │
│  │ Acquire  │       ⑦ 探头（Probe）                  │
│  └──────────┘       ┌──────────┐                    │
│                     │  尖头    │                    │
│                     │  ──┤├── │                    │
│                     │  BNC头   │                    │
│                     └──────────┘                    │
└─────────────────────────────────────────────────────┘
各部分说明：
① 显示屏幕
- 现代示波器多为彩色 LCD 屏
- 屏幕上有 网格线（Grid），通常横竖各 10 条线，将屏幕分成 10 × 8 格（Division，简称 div）
- 横轴 = 时间，纵轴 = 电压
- 屏幕上会显示当前通道、时基、电压比例等信息
> 重点概念：格（Division/div）
> 示波器屏幕上的一格不是固定代表多少电压或多少时间，它取决于你的设置。比如：
> - 如果设置 V/div = 1V，那么一格的高度代表 1V
> - 如果设置 Time/div = 1ms，那么一格的宽度代表 1ms
> 这个"格"的概念后面会反复用到，一定要记住。
② 水平控制区（Horizontal）
- Time/Div（时基）旋钮：控制屏幕上每格代表多少时间
  - 顺时针旋转 → 时间变短 → 波形在屏幕上被"拉伸"（看细节）
  - 逆时针旋转 → 时间变长 → 波形在屏幕上被"压缩"（看全局）
- 常见范围：1ns/div ~ 50s/div
③ 垂直控制区（Vertical）
- V/Div（电压比例）旋钮：控制屏幕上每格代表多少电压
  - 顺时针旋转 → 电压变大 → 波形在屏幕上被"压缩"
  - 逆时针旋转 → 电压变小 → 波形在屏幕上被"拉伸"
- 每个通道（CH）有自己独立的 V/div 旋钮
- 常见范围：1mV/div ~ 10V/div
④ 触发控制区（Trigger）
- Trigger Level（触发电平）旋钮：设置在哪个电压值上"抓住"波形
- 触发是示波器最核心的概念之一，我们会在 模块 5 详细讲解
- 简单理解：触发就像相机的快门——告诉示波器"在这个条件下开始显示波形"
⑤ 功能按键区
- Measure（测量）：自动测量频率、Vpp、周期等参数
- Cursor（光标）：手动测量两点之间的电压差或时间差
- Math（数学运算）：对波形进行加减乘除、FFT 等运算
- Acquire（采集）：设置采样模式（普通、峰值检测、平均等）
- Display（显示）：调整显示方式（余辉、色温等）
⑥ 信号输入接口（BNC 接口）
- BNC（Bayonet Neill-Concelman） 接口：示波器上用来连接探头的同轴连接器
- 常见通道数：2 通道（入门级）、4 通道（主流）、更多（高端）
- 每个通道对应一个 CH 按钮，按一下点亮该通道，再按一下关闭
⑦ 探头（Probe）
- 探头是连接被测电路和示波器的"桥梁"
- 探头的详细讲解在 模块 6，现在只需要知道它的基本作用
- 探头一端是 BNC 接口（插到示波器上），另一端是 探针尖端（接触被测电路）

1.5 一个直观的理解：示波器在做什么？
想象你站在一条河边，河水的水位在上下波动：
水位
 ↑
 │    ╱╲      ╱╲      ╱╲
 │   ╱  ╲    ╱  ╲    ╱  ╲
 │  ╱    ╲  ╱    ╲  ╱    ╲
 │ ╱      ╲╱      ╲╱      ╲
 │╱                          ╲
 └──────────────────────────────→ 时间
- Y 轴（垂直）= 水位高度 = 电压
- X 轴（水平）= 时间流逝 = 时间
- 示波器就是这样一个"水位记录仪"，只不过它记录的不是河水，而是电路中的电压变化

1.6 模块 1 小结
知识点	核心内容
示波器是什么	将电信号随时间变化以波形图形显示的仪器
示波器 vs 万用表	示波器看动态（波形），万用表看静态（数值）
格（Division）	屏幕的基本单位，横 10 格 × 纵 8 格，每格代表的值由设置决定
Time/Div	每格代表多少时间，控制波形的水平缩放
V/Div	每格代表多少电压，控制波形的垂直缩放
BNC 接口	探头与示波器的连接器
通道（Channel）	示波器的输入通道，每通道可独立测量一个信号
