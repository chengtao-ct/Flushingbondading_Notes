## 核心要点

> 每次使用前做探头补偿，接地夹必须接 GND，接地线尽量短。这三件事决定了你看到的波形是否真实。

---

## 1. 探头连接

### 物理结构

```
探针尖端（接触被测引脚）──→ 有钩子，可翻开钩住引脚
    │
同轴电缆
    │
探头体 ──→ 10X/1X 拨动开关
    │
    ├── BNC 接口（插示波器）：对准卡槽，推入顺时针转 1/4 圈
    │
    └── 接地夹（鳄鱼夹）：必须夹到被测电路板的 GND
```

### 连接步骤

1. 探头 BNC 接口插入示波器 CH 通道（推入 + 顺时针旋转锁紧）
2. 翻开探针尖端钩子，**钩住**被测引脚
3. 接地夹夹到被测板最近的 **GND** 点
4. 轻拉确认钩紧、夹紧

### 接地夹夹哪里？

| 方式 | 优先级 |
|:---|:---|
| 板上标 GND 的排针/测试点 | 最推荐 |
| 电解电容负极 | 次选 |
| USB 口金属外壳 | 备选 |
| 板边裸露铜皮 | 最后选择 |

> **所有探头的接地夹必须接同一个 GND！** 示波器内部各通道地是连通的，接不同地可能造成短路。

## 2. 探头补偿操作（每次使用前必做）

### 步骤

```
① 探头钩住示波器面板上的 CAL 端子
② 接地夹夹 CAL 旁的 GND 端子
③ V/div → 1V，Time/div → 1ms，触发 → CH1 上升沿
④ 观察屏幕上的方波
⑤ 用绝缘小螺丝刀插入探头 BNC 端附近的小孔
⑥ 旋转补偿电容，直到方波角是平的
```

### 判断标准

| 看到的形状 | 状态 | 操作 |
|:---|:---|:---|
| 角变圆（边沿弧形） | 欠补偿 | 顺时针转 |
| 角变尖（出现过冲） | 过补偿 | 逆时针转 |
| **角是平的（方方正正）** | **正确** | **搞定** |

> **口诀：圆→顺时针，尖→逆时针，平→不用调。**

## 3. 10X/1X 设置

### 探头端

拨动探头体上的开关到 **10X**。

### 示波器端（容易忘！）

1. 按 CH1 按钮
2. 菜单中找到"探头"或"Probe"
3. 选择 **10X**

> 如果忘记设，电压读数会**差 10 倍**（比如实际 3.3V 显示为 0.33V 或 33V）。
> 部分智能探头会自动识别，但旧款/便宜探头需要手动设置。

## 4. 高频测量接地技巧

| 方法 | 接地线长度 | 适用 |
|:---|:---|:---|
| 鳄鱼夹夹最近 GND | ~10cm | 低频（< 1MHz） |
| **接地弹簧**（探头配件） | ~1-2cm | 高频（> 20MHz） |
| PCB 焊盘弹簧 | ~0.5cm | 专业高频测量 |

> **接地线越短越好。** 20cm 以上的接地线会让高频信号出现严重振铃。
