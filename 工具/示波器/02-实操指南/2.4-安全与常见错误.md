## 核心要点

> 最危险的操作是**接地短路**——示波器地线通过电源连大地，接地夹接错地方可能烧毁电路或示波器。测市电必须用隔离或差分探头。

---

## 1. 安全红线

### 绝对不要做的事

| 行为 | 后果 |
|:---|:---|
| **用普通探头直接测市电** | 可能短路、触电、烧示波器 |
| **接地夹夹到非 GND 的点** | 通过示波器地线短路到大地，烧电路 |
| **两个探头接不同的非等电位"地"** | 短路 |
| **50Ω 输入模式下输入 >5V 信号** | 烧毁示波器前端 |
| **测量时手触探针尖端** | 引入人体噪声，高压时触电 |

### 测市电的正确方法

| 方法 | 安全性 | 说明 |
|:---|:---|:---|
| **差分探头** | ✅ 安全 | 必须用高压差分探头 |
| **隔离示波器** | ✅ 安全 | 电池供电、通道隔离的示波器 |
| **100X 高压探头** | ⚠️ 需谨慎 | 确认探头额定电压足够 |
| **普通探头直接测** | ❌ 禁止 | 接地夹会短路火线到大地！ |

> **为什么普通探头不能测市电？** 示波器地线通过电源线接大地。市电的零线在变压器端接大地。接地夹夹到火线 → 火线通过地线直连大地 → 短路爆炸。

## 2. 新手常见错误

| 错误 | 症状 | 解决 |
|:---|:---|:---|
| 不做探头补偿 | 方波有圆角或尖角 | 用 CAL 信号做补偿 |
| 忘设探头 10X | 电压读数差 10 倍 | CH 菜单 → 探头 → 10X |
| 接地夹不接 | 波形乱跳或完全不对 | 必须接 GND |
| 触发电平设错 | 波形左右跑或屏幕不动 | 调 Level 到波形中间 |
| Time/div 不对 | 看不到完整波形或太密 | 调到显示 2~5 个周期 |
| 用 1X 测高频 | 方波边沿变圆 | 切到 10X |
| 用正弦插值看方波 | 出现假振铃 | 切换到线性插值或 Dots |
| 通道没打开 | 屏幕上看不到波形 | 按 CH 按钮确认 LED 亮 |

## 3. 故障排查速查表

### 波形显示问题

| 现象 | 检查项 | 解决 |
|:---|:---|:---|
| 完全没有波形 | 通道是否开启？探头是否插紧？信号是否存在？ | 按 CH 键、重插探头、确认有信号 |
| 波形乱跑 | 触发设置是否正确？ | 调触发电平到波形中间 |
| 波形太矮 | V/div 是否太大？ | 逆时针转 Scale |
| 波形超出屏幕 | V/div 是否太小？ | 顺时针转 Scale |
| 波形太密 | Time/div 是否太大？ | 顺时针转 Horizontal Scale |
| 看不到周期 | Time/div 是否太小？ | 逆时针转 Horizontal Scale |
| 屏幕定格不动 | 是否 Normal/Single 模式？触发条件是否满足？ | 切 Auto，调触发电平 |
| 有奇怪的振铃 | 探头补偿做了吗？接地线长不长？ | 做补偿，缩短接地线 |
| 噪声很大 | 带宽限制开了吗？接地是否良好？ | 开 20MHz 限制，检查接地 |

### 测量值异常

| 现象 | 检查项 |
|:---|:---|
| 电压差 10 倍 | 探头衰减比设置是否与实际一致 |
| 频率读数不对 | 可能混叠——改变 Time/div 看频率是否变化 |
| 占空比不对 | 确认测量的是完整周期 |

## 4. 日常使用注意事项

| 注意事项 | 原因 |
|:---|:---|
| 先断探头，再动板子 | 防止钩子拉弯引脚 |
| 接地夹必须夹紧 | 松动的地线是噪声头号来源 |
| 每次只测一组信号 | 减少出错概率 |
| 测完就记录 | 不要等全测完再回忆 |
| 用完关闭通道 | 减少干扰 |
| 定期做探头补偿 | 保证测量精度 |
