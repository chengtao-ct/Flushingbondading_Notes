## 核心要点

> 三种测量方式：**读网格**（快速估算）、**自动测量**（按 Measure）、**光标测量**（按 Cursor，最精确）。

---

## 1. 自动测量（Measure）

按 **Measure** 按钮 → 添加测量项。

### 常用测量项

| 类别 | 可测参数 | 说明 |
|:---|:---|:---|
| **电压** | Vpp, Vmax, Vmin, Vtop, Vbase, Vamp, Vavg, Vrms, Overshoot | 最常用：Vpp、Vmax、Vmin |
| **时间** | Period, Frequency, Rise Time, Fall Time, Pulse Width, Duty Cycle | 最常用：Freq、Period、Rise Time |
| **延迟** | Phase（相位差）, Delay（时间差） | 需要双通道 |

### 操作步骤

1. 按 **Measure**
2. 选择通道（CH1/CH2）
3. 选择要测量的参数
4. 屏幕上会显示测量值
5. 最多可同时显示 4~6 个参数（取决于示波器型号）

---

## 2. 光标测量（Cursor）

按 **Cursor** 按钮 → 选择光标类型。

### 电压光标

测量两点之间的**电压差（ΔV）**。

```
操作：
1. Cursor → 类型 → 电压
2. 屏幕出现两条水平虚线（光标A、光标B）
3. 旋转旋钮将光标A拖到波形最高点
4. 按 A↔B 切换，将光标B拖到波形最低点
5. 屏幕显示 ΔV = V_A - V_B

例：光标A在 3.3V，光标B在 0V
   → ΔV = 3.3V = Vpp
```

### 时间光标

测量两点之间的**时间差（ΔT）**和**频率**。

```
操作：
1. Cursor → 类型 → 时间
2. 屏幕出现两条垂直虚线（光标A、光标B）
3. 将光标A拖到一个上升沿
4. 将光标B拖到下一个上升沿
5. 屏幕显示 ΔT 和 1/ΔT（频率）

例：ΔT = 50μs → f = 20 kHz
```

### 光标操作技巧

| 操作 | 方法 |
|:---|:---|
| 切换光标A/B | 按 Cursor 菜单中的 A↔B 或 Source 切换 |
| 移动光标 | 旋转旋钮（通常是多用途旋钮） |
| 同时移动两条光标 | 有些示波器支持锁定模式 |
| 光标跟踪 | 有些示波器支持光标自动跟踪波形 |

## 3. Math 运算

按 **Math** 按钮。

### 常用运算

| 运算 | 公式 | 应用 |
|:---|:---|:---|
| **CH1 - CH2** | 两通道相减 | 等效差分测量（非精确） |
| **CH1 + CH2** | 两通道相加 | 功率分析（电压+电流） |
| **CH1 × CH2** | 两通道相乘 | 瞬时功率（V×I） |
| **FFT** | 快速傅里叶变换 | 频域分析 |

### FFT（频域分析）

将时域信号转换到频域，观察信号的**频率成分**。

```
操作：
1. Math → 运算 → FFT
2. 选择源通道（如 CH1）
3. 设置窗函数（通常用 Hanning）
4. 屏幕下半部分显示频谱图

看什么：
- 信号基频的幅度
- 谐波分量（2次、3次...）
- 底噪水平
- 有无异常频率成分
```

**FFT 应用场景：**
- 分析电源噪声的频率成分
- 查看 PWM 信号的谐波分布
- 确认时钟信号的频谱纯度
- 诊断 EMI 问题

---

## 4. 三种测量方式对比

| 方式 | 操作 | 精度 | 速度 | 适用 |
|:---|:---|:---|:---|:---|
| **读网格** | 数格 × V/div 或 Time/div | 低（±1格） | 最快 | 快速估算 |
| **自动测量** | Measure → 选参数 | 中 | 快 | 日常调试 |
| **光标测量** | Cursor → 拖光标 | 高 | 较慢 | 精确分析 |

---

## 5. 测量技巧

| 技巧 | 说明 |
|:---|:---|
| 先用自动测量确认大致值 | 再用光标精测 |
| 多次读数取平均 | 减少随机误差 |
| 波形不稳定时不要测 | 先调触发再测量 |
| 用 Stop 冻结波形 | 需要仔细分析时先冻结 |
| 平均模式去噪 | Acquire → 平均 → 选次数（重复信号适用） |
