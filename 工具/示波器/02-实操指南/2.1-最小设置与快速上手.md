## 核心要点

> 坐到示波器前，按以下流程操作即可。重点是：探头 10X、耦合 DC、波形占 3~6 格、触发电平在信号中间。

---

## 最小设置清单（照着做）

| 步骤 | 操作 | 备注 |
|:---:|------|------|
| 1 | 探头开关拨到 **10X** | 探头体上的拨动开关 |
| 2 | 示波器 CH 菜单 → 探头选 **10X** | 否则电压差 10 倍！ |
| 3 | 耦合设为 **DC** | CH 菜单 → 耦合 → DC |
| 4 | V/div → **1V**（3.3V信号）或 **2V**（5V信号） | 波形占 3~6 格即可 |
| 5 | Time/div → **10~20μs**（看 PWM） | 显示 2~5 个完整周期 |
| 6 | 触发源 → **CH1**（信号所在通道） | Trigger Menu → Source |
| 7 | 触发沿 → **上升沿** | Trigger Menu → Slope → Rising |
| 8 | 触发电平 → **~1.5V**（3.3V 信号） | 旋转 Level 旋钮，让触发线在波形中间 |
| 9 | 触发模式 → **Auto** | 日常调试用 Auto |

**不确定时按 Auto Set**，示波器会自动设置，然后再手动微调。

## 不同信号的推荐设置

| 信号 | V/div | Time/div | 触发电平 |
|:---|:---|:---|:---|
| 3.3V 方波/PWM | 1V | 10~20μs（20kHz PWM） | 1.5V |
| 5V 方波 | 1~2V | 10~20μs | 2~2.5V |
| 3.3V UART (115200) | 1V | 10μs | 1.5V |
| I2C (400kHz) | 1V | 1~2μs | 1.5V |
| SPI (10MHz) | 1V | 100ns | 1.5V |
| 电源纹波（AC耦合） | 10~50mV | 1~10ms | 设在纹波中间 |
| 上电过程 | 2~5V | 500ms~1s | 1~2V |

## 垂直系统操作速查

### V/div（Scale 旋钮）

- 顺时针 → 电压变大/格 → 波形压缩
- 逆时针 → 电压变小/格 → 波形拉伸
- **按一下旋钮**切换粗调/细调
- 1-2-5 序列：1mV → 2mV → 5mV → 10mV → ... → 1V → 2V → 5V → 10V

### Position 旋钮

- 上下移动波形，不改变比例
- 用途：分开重叠的通道、对齐网格线

### 耦合

| 耦合 | 用途 |
|:---|:---|
| **DC**（默认） | 看全部信号（直流+交流），日常使用 |
| **AC** | 只看交流分量，适合看电源纹波、小噪声 |
| **GND** | 断开输入显示 0V 基线，用于校准零点位置 |

### 带宽限制

- CH 菜单 → 带宽限制 → **20MHz**
- 效果：滤掉高频噪声，波形更干净
- 什么时候用：PWM/电机信号、电源纹波、低速数字电路
- 快速判断技巧：开启后异常消失 → 不是信号问题，是噪声

## 水平系统操作速查

### Time/div（Horizontal Scale 旋钮）

- 顺时针 → 时间变短/格 → 波形拉伸 → 看细节
- 逆时针 → 时间变长/格 → 波形压缩 → 看全局
- 同样是 1-2-5 序列

### Position 旋钮

- 左右移动波形
- 左旋 → 看更多触发后的数据
- 右旋 → 看更多触发前的数据

## 完整操作流程

```
① 开机 → 等待初始化
② 探头补偿（钩 CAL 端子，调到方波平角）
③ 探头钩被测点，接地夹夹最近的 GND
④ CH 菜单：耦合 DC，探头 10X
⑤ 旋转 V/div → 波形占 3~6 格
⑥ 旋转 Time/div → 显示 2~5 个周期
⑦ 触发设置：源=信号通道，沿=上升沿，Level=信号50%
⑧ 确认波形稳定 → 开始测量
```

## 出问题了看这里

| 现象 | 原因 | 解决 |
|:---|:---|:---|
| 波形左右乱跑 | 触发电平不在信号范围内 | 调 Level 到波形中间 |
| 波形太小看不清 | V/div 设大了 | 逆时针转 Scale |
| 波形超出屏幕 | V/div 设小了 | 顺时针转 Scale |
| 看不到完整周期 | Time/div 设小了 | 逆时针转 Horizontal Scale |
| 波形太密看不清 | Time/div 设大了 | 顺时针转 Horizontal Scale |
| 屏幕不动了 | Normal 模式触发条件不满足 | 切 Auto，检查触发电平 |
| 电压读数不对 | 探头衰减比没设对 | CH 菜单 → 探头 → 10X |
| 波形有奇怪的振铃 | 探头没补偿或接地线太长 | 做补偿、缩短接地线 |
